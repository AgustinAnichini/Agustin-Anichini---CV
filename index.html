Proyectos

Proyecto mascotas - UNLaM

Desarrollé una aplicación web orientada a ayudar a los dueños a encontrar sus mascotas perdidas. Aplicamos metodología SCRUM y diseño responsivo para asegurar accesibilidad y funcionalidad en diferentes dispositivos. La plataforma permite a los usuarios publicar información sobre mascotas perdidas y encontradas, facilitando la comunicación entre personas que han perdido o encontrado una mascota.
Durante el desarrollo del proyecto, adquirí valiosas habilidades técnicas, además de mejorar en investigación, trabajo en equipo y comunicación.

Proyecto DriveDoctor - UNLaM

DriveDoctor es una aplicación diseñada para ayudar a los usuarios a diagnosticar y gestionar fallas técnicas en sus vehículos. La plataforma permite a los usuarios ingresar síntomas del vehículo y obtener posibles diagnósticos y recomendaciones de mantenimiento. Utilizamos metodología SCRUM y principios SOLID para asegurar un desarrollo ágil y estructurado, y realizamos pruebas unitarias y E2E para garantizar la calidad del producto.
Este proyecto me brindó la oportunidad de fortalecer mis competencias técnicas y de liderazgo dentro de un marco de desarrollo ágil.

Sabiondos

Lideré el desarrollo de Sabiondos, un juego educativo innovador, como parte de una asignatura de mi carrera. Utilizamos PHP con arquitectura MVC, incorporando tecnologías como Mustache para plantillas, PHPMailer para correos, phpqrcode para códigos QR, y jpgraph para visualización de datos. Diseñé e implementé eficientemente la base de datos, gestionando preguntas, respuestas, usuarios y estadísticas del juego.
Esta experiencia enriqueció mis habilidades técnicas y de gestión de proyectos, permitiéndome trabajar eficazmente en un entorno colaborativo y desafiante.

E-commerce

Esta experiencia me permitió profundizar en conceptos avanzados de JavaScript, así como en tecnologías complementarias, fortaleciendo mi capacidad para crear soluciones web completas y funcionales.
